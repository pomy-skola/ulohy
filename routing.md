# Routing

*odhadovaný čas: ~35 min. + přečtení zadání*

Vytvoříte aplikaci postavenou na frameworku [Flask](https://flask.palletsprojects.com/).

Založte pro aplikaci repozitář na serveru Github, kam umístíte zdrojové kódy.

Připravte si výchozí stránku stejně, jako v úkolu [Integrace frameworku bootstrap do HTML stránky](./jinja-template.md).



Úpravy:
- Odstraňte formulář pro hledání (vpravo nahoře)
- Upravte název v nadpisu H1 na název úlohy na Ámosovi.
- Upravte modrý odkaz tak, aby směroval na váš Github repozitář.
- Místo textu "This example ..." vložte aktuální datum a čas.
- Do navigační lišty přidejte odkaz na stránku "O mně". Stránka bude obsahovat vaše jméno, předmět, třídu a ročník (v tabulce).

Do řešení úkolu vložte link na repozitář.
